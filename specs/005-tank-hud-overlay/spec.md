# Feature Specification: 水槽HUDオーバーレイ

**Feature Branch**: `005-tank-hud-overlay`
**Created**: 2026-03-24
**Status**: Draft
**Input**: User description: "水槽のシーンの上部に経過時間を表示してポモドーロ時間を過ぎると赤色になるようにしたい。時間表示もドット絵調にして。また右にドット絵風のコインと現在のpomoの数字を表示したい。クリックでviewを開くのではなく小さい画面でも基本的な機能は全てできるようにして拡大表示用のボタンをドット絵調で作りたい"

## Clarifications

### Session 2026-03-25

- Q: コンパニオンビューでアクション実行時のフィードバック方法は？ → A: ボタンにドット絵調のビジュアルフィードバック（発光・色変化）＋獲得ポモ数のHUD上アニメーション
- Q: フルサイズ水槽パネルもHUDオーバーレイスタイルに変更するか？ → A: フルサイズパネルも新しいHUDオーバーレイスタイルに統一する（既存StatsBarを置き換え）

## User Scenarios & Testing *(mandatory)*

### User Story 1 - ポモドーロ経過時間の常時確認 (Priority: P1)

ユーザーはコンパニオンビュー・フルサイズパネル両方の水槽シーン上部にドット絵調で描かれた経過時間タイマーを常時確認できる。タイマーは最後のメンテナンスアクション（餌やり・水替え・苔掃除）からの経過時間を表示する。ポモドーロの目安時間（20分）を過ぎるとタイマーの色が赤に変化し、休憩タイミングを視覚的に知らせる。

**Why this priority**: タイマー表示はポモドーロタイマーとしての核心機能であり、ユーザーが最も頻繁に確認する情報。赤色変化により「そろそろ休憩」の合図を直感的に伝える。

**Independent Test**: コンパニオンビューを表示し、経過時間が増加していくこと、20分超過で赤色に変わることを目視で確認できる。

**Acceptance Scenarios**:

1. **Given** コンパニオンビューが表示されている, **When** ポモドーロセッションが進行中, **Then** 水槽シーン上部にドット絵調フォントで経過時間が `mm:ss` 形式で表示される
2. **Given** 経過時間が20分未満, **When** タイマーを確認する, **Then** タイマーは通常色（白系）で表示される
3. **Given** 経過時間が20分以上, **When** タイマーを確認する, **Then** タイマーが赤色に変化している
4. **Given** ユーザーがメンテナンスアクションを実行した, **When** アクション完了後, **Then** タイマーが00:00にリセットされる
5. **Given** ライトがオフ, **When** タイマーを確認する, **Then** タイマーが一時停止していることが視覚的にわかる

---

### User Story 2 - コンパニオンビューでの基本操作 (Priority: P1)

ユーザーは小さなコンパニオンビュー（サイドバー）から水槽パネルを開かずに全ての基本操作（餌やり・水替え・苔掃除・ライト切替）を行える。現在の「クリックでパネルを開く」挙動を廃止し、コンパニオンビュー単体で完結する操作体験を提供する。

**Why this priority**: 小さい画面で基本機能を完結させることはユーザー体験の根幹であり、拡大ビューへの依存を解消する最重要改善。

**Independent Test**: コンパニオンビューのみで餌やり・水替え・苔掃除・ライト切替を実行し、状態が正しく反映されることを確認できる。

**Acceptance Scenarios**:

1. **Given** コンパニオンビューが表示されている, **When** ユーザーがアクションボタン（餌やり等）をタップ/クリックする, **Then** 対応するアクションが実行され水槽の状態が更新される
2. **Given** コンパニオンビューが表示されている, **When** 各アクションボタンを確認する, **Then** ドット絵調のアイコンで餌やり・水替え・苔掃除・ライト切替のボタンが表示されている
3. **Given** 水槽が餌やり不要な状態, **When** 餌ボタンを確認する, **Then** ボタンが無効化されているか視覚的に不要であることがわかる
4. **Given** ユーザーがアクションボタンをクリックした, **When** アクションが成功する, **Then** ボタンがドット絵調のビジュアルフィードバック（発光・色変化）で応答し、獲得ポモ数がHUD上にアニメーション表示される
5. **Given** ユーザーがアクションボタンをクリックした, **When** アクションがポモを獲得する, **Then** HUDのコイン表示付近に獲得数の浮遊アニメーションが表示され、残高が更新される

---

### User Story 3 - ポモコイン残高の常時表示 (Priority: P2)

ユーザーは水槽シーン上部の右側にドット絵風のコインアイコンと現在のポモ残高を常時確認できる。ストアで使えるポモの数がひと目でわかる。

**Why this priority**: ポモ残高は重要なゲーム情報だが、タイマーや基本操作ほど頻繁に参照しない。表示自体はシンプルで実装リスクも低い。

**Independent Test**: コンパニオンビューを表示し、コインアイコンと数字が正しく表示されること、アクション実行後に数値が更新されることを確認できる。

**Acceptance Scenarios**:

1. **Given** コンパニオンビューが表示されている, **When** HUDエリアを確認する, **Then** ドット絵風コインアイコンと現在のポモ残高が右側に表示されている
2. **Given** ポモ残高が変化するアクションを実行した, **When** HUDを確認する, **Then** 表示されるポモ残高が更新されている

---

### User Story 4 - 拡大表示ボタン (Priority: P2)

ユーザーはコンパニオンビューに配置されたドット絵調の拡大ボタンをクリックして、詳細な水槽パネル（ストア・詳細ステータス等）を開ける。水槽シーン全体のクリックではなく、明示的なボタンで拡大操作を行う。

**Why this priority**: 基本操作がコンパニオンビューで完結した上で、詳細情報やストアへのアクセス手段として必要。

**Independent Test**: 拡大ボタンをクリックして水槽パネルが開くことを確認できる。水槽シーンの他の部分をクリックしてもパネルが開かないことを確認できる。

**Acceptance Scenarios**:

1. **Given** コンパニオンビューが表示されている, **When** ドット絵調の拡大ボタンをクリックする, **Then** 水槽パネル（フルサイズ）が開く
2. **Given** コンパニオンビューが表示されている, **When** 水槽シーンの拡大ボタン以外の領域をクリックする, **Then** 水槽パネルは開かない（従来の全体クリック挙動を廃止）

---

### Edge Cases

- 経過時間が99:59を超えた場合、表示は `99:59+` にキャップする
- ライトオフ中はライト切替ボタンのみ有効とし、他のアクションボタンは無効化する
- コンパニオンビューの幅が極端に狭い場合、最小幅を設定しHUD要素が重ならないようにする
- ポモ残高が非常に大きい数値（10000以上）の場合は `9999+` 形式で省略表示する

## Requirements *(mandatory)*

### Functional Requirements

- **FR-001**: システムは水槽シーン上部にドット絵調フォントで経過時間を `mm:ss` 形式で表示しなければならない
- **FR-002**: システムは経過時間が20分（ポモドーロ目安時間）を超えた場合、タイマー表示を赤色に変更しなければならない
- **FR-003**: システムはメンテナンスアクション実行時にタイマーを00:00にリセットしなければならない
- **FR-004**: システムはライトオフ中にタイマーを一時停止し、一時停止状態を視覚的に示さなければならない
- **FR-005**: システムはHUD右側にドット絵風コインアイコンと現在のポモ残高を表示しなければならない
- **FR-006**: システムはコンパニオンビュー内にドット絵調のアクションボタン（餌やり・水替え・苔掃除・ライト切替）を配置しなければならない
- **FR-007**: 各アクションボタンはコンパニオンビューから直接アクションを実行できなければならない（水槽パネルを開かずに）
- **FR-008**: システムはドット絵調の拡大ボタンを提供し、クリックで水槽パネル（フルサイズ）を開けなければならない
- **FR-009**: 水槽シーン全体のクリックによるパネル表示の既存挙動を廃止し、拡大ボタンのみでパネルを開く方式に変更しなければならない
- **FR-010**: アクション不要な状態（空腹度が低い等）のボタンは無効化状態を視覚的に表示しなければならない
- **FR-011**: アクション成功時、ボタンはドット絵調のビジュアルフィードバック（発光・色変化）で応答しなければならない
- **FR-012**: ポモ獲得時、HUDのコイン表示付近に獲得数の浮遊アニメーションを表示しなければならない
- **FR-013**: フルサイズ水槽パネルの既存StatsBarをHUDオーバーレイスタイルに置き換え、コンパニオンビューと統一されたビジュアルを提供しなければならない

### Key Entities

- **HUDオーバーレイ**: 水槽シーン上部に重ねて表示される情報バー。経過時間（左）とポモ残高（右）を含む
- **アクションボタン群**: 水槽シーン下部に配置されるドット絵調の操作ボタン。餌やり・水替え・苔掃除・ライト切替・拡大の5種
- **ドット絵調タイマー**: ピクセルアートスタイルでレンダリングされた時刻表示。通常は白系、20分超過で赤色に変化

## Success Criteria *(mandatory)*

### Measurable Outcomes

- **SC-001**: ユーザーはコンパニオンビューのみで全ての基本メンテナンス操作（餌やり・水替え・苔掃除・ライト切替）を完了できる
- **SC-002**: 経過時間タイマーは1秒単位で更新され、実際の経過時間と1秒以内の誤差で一致する
- **SC-003**: ポモドーロ目安時間（20分）超過から1秒以内にタイマー色が赤に変化する
- **SC-004**: ポモ残高表示はアクション実行後の状態更新と同時に反映される
- **SC-005**: 拡大ボタン以外の水槽シーンクリックではパネルが開かない（誤操作の防止）
- **SC-006**: 全てのHUD要素とボタンがドット絵調の統一されたビジュアルスタイルで表示される

## Assumptions

- ポモドーロの目安時間は既存のブレイクウィンドウ開始時間（20分）を採用する
- ドット絵調フォントはピクセル単位で描画し、既存の水槽シーンと同じレンダリング方式を踏襲する
- コンパニオンビューの表示領域は現状サイズを維持し、HUDとボタンをこの制約内に収める
- ストア機能は引き続き水槽パネル（フルサイズ）でのみ利用可能とする
- フルサイズパネルの既存StatsBarはHUDオーバーレイに完全に置き換える（StatsBarの情報はHUD内に統合）
